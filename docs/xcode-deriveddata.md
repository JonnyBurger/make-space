## Available platforms

- macOS

## Command

```sh
rm -rfv ~/Library/Developer/Xcode/DerivedData
```

## Description

Deletes the 'DerivedData' folder from Xcode.
You don't necessarily need these files as they are being regenerated during Xcode build processes.
